# 交通タブ 設計spec

**日付**: 2026-03-19
**フェーズ**: Phase 5 M1

---

## 概要

有明エリア共通の交通情報（時刻表）を独立タブとして提供する。イベントや施設に依存しない共通コンテンツとして、現在時刻以降の発車時刻を一覧表示する。

---

## 配置

- トップレベルのビュータブに「交通」を追加
- 既存タブ: 今日 / 今週 / カレンダー / **交通**（新規）
- イベントモーダル内には配置しない（後で変更可能な設計とする）

---

## データ設計

### `packages/web/src/data/timetable.ts`

路線ごとに平日ダイヤ・土休日ダイヤを静的定義。

```ts
type Departure = string; // "HH:MM"

type DirectionSchedule = {
  label: string;       // 方面名（例: "大崎方面"）
  weekday: Departure[];
  holiday: Departure[];
};

type RouteData = {
  name: string;           // 路線名（例: "りんかい線"）
  station: string;        // 最寄り駅・停留所名
  walkMinutes: number;    // 徒歩分数
  directions: DirectionSchedule[];
};

export const timetable: RouteData[] = [...];
```

**対象路線（4路線）:**

| 路線 | 最寄り | 方面 |
|---|---|---|
| りんかい線 | 有明テニスの森駅 | 大崎方面 / 新木場方面 |
| ゆりかもめ | 有明駅 | 新橋方面 / 豊洲方面 |
| 都バス | 有明停留所 | 複数路線・複数方面（実データ確認後に確定） |
| BRT | 有明BRT停留所 | 新橋行 / 晴海行 |

**ダイヤ更新**: ダイヤ改正時のみ手動更新。GitHub Actionsで毎年3月1日・10月1日にリマインダーIssueを自動作成する（対応後にcloseするだけ）。

- りんかい線: 3月固定（JR東日本春改正連動）
- ゆりかもめ: 3月 or 10月（不定）
- 都バス: 3〜4月・9〜10月が最多
- BRT: 4月・9〜10月が多い（拡張フェーズにつき随時）

## UI設計

### 表示形式

横スクロール対応テーブル。列構造:

```
路線名（colspan=方面数）
最寄り駅・停留所
ここから徒歩XX分
方面名（1列1方向）
時刻（現在時刻以降、縦に列挙）
```

### 時刻フィルタリング

- 今日が平日か土休日かをブラウザのDateで自動判定
- 現在時刻以降の便のみ表示（切り替えなし）
- 直近の便から順に表示

### ファイル構成

```
packages/web/src/
  data/
    timetable.ts       # 時刻表静的データ
  components/
    TransportView.tsx  # 交通タブビューコンポーネント
  lib/
    timetableUtils.ts  # 平日/土休日判定、現在時刻フィルタ
```

## スコープ外（別フェーズ候補）

- 遅延情報アラート（ODPT API）
- 「帰りを調べる」外部リンク
- イベントモーダル内への交通情報埋め込み

---

## 変更容易性

- `timetable.ts` を独立ファイルとして持つことで、表示場所の変更（タブ→モーダル等）はコンポーネント側のみの修正で対応可能
- 路線追加・削除は `timetable.ts` の配列操作のみ
